Troubleshooting

Bot not responding?

  • Check bot has "Send Messages" permission in the channel
  • Make sure bot role is above other roles
  • Check bot is online (see status widget on this page)

Commands not showing up?

  • Wait a few minutes after inviting bot
  • Try kicking and re-inviting the bot
  • Make sure you granted "applications.commands" permission

Can't add strains?

  • Only server owners can use /addstrain
  • All fields except image are required
  • Make sure percentages are in valid ranges
